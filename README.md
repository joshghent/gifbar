## Running the app locally

```shell
$ yarn start
```

## Developing

_Watch All_
```shell
$ yarn watch
```

_Electron_
```shell
$ yarn watch:electron
```

_React_
```shell
$ yarn watch:react
```

_Create DMG_
```shell
$ yarn run package-dmg
```

_Create App_
```shell
$ yarn run package
```
